# MicroMouse,

## January 2019 - March 2019,

- [github](https://github.com/ec965/micromouse_2019)

MicroMouse is a world wide competition in the promotion of autonomous vehicles.
Using IR transceivers to detect distance from walls, the mouse can navigate
through a maze using it's microcontroller "brain".
